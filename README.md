Magenta-cyan porcelain portal. Logo-only visuals, kinetic type, ridiculous but clean.

## Stack
- Next.js 14 with App Router
- TypeScript
- Tailwind CSS with custom magenta-cyan theme
- Framer Motion for animations
- Full accessibility compliance

## Development

```bash
npm install
npm run dev
```

## Deployment

This site auto-deploys to Vercel from the `main` branch.

Live at: [cosmictoilet.xyz](https://cosmictoilet.xyz)

## Content Management

- Edit text in `/content/cosmictoilet/strings.json`
- Update links in `/content/cosmictoilet/links.json`  
- Modify colors in `/theme/cosmictoilet.ts`
- Markdown content in `/content/cosmictoilet/`
